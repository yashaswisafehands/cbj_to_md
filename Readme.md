# How to Run the Project

1. **Clone the repository:**
    ```sh
    git clone https://github.com/yashaswisafehands/cbj_to_md.git
    cd cbj_to_md
    ```

2. **Install dependencies:**
    ```sh
    pip install -r requirements.txt
    ```

3. **Run the project:**
    ```sh
    python cbj_md.py
    ```
